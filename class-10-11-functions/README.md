# Class 10-11 Functions

This folder contains solutions for Set A of the Eid Break Assignment.

## Files

### calculator.js

All operations are implemented as pure functions.

Contains:

- Addition
- Subtraction
- Multiplication
- Division
- Square Root
- Power
- Factorial

### validators.js

Contains validation functions for:

- Email
- Phone Number
- Strong Password
- URL
- Date (YYYY-MM-DD)

## Running The Files

Calculator

```bash
node calculator.js
```

Validation Library

```bash
node validators.js
```
