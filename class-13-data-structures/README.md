# Class 13 - Data Structures

This folder contains solutions related to data structures, arrays, objects, sorting, filtering, and data manipulation.

## Projects

### Dynamic Table

A dynamic employee table built with HTML, JavaScript, and Tailwind CSS.

#### Features

- Display employee data in a table
- Search employees by name
- Sort by:
  - ID
  - Name
  - Department
  - Salary
- Pagination (10 records per page)
